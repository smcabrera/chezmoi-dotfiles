# Pantry Manager Skill Triggers

This skill should be activated when the user asks to:

## Primary Triggers
- "pantry" (in the context of managing food/ingredients)
- "meal plan"
- "meal planning"
- "ingredient reuse"
- "recipe planning"

## Command Triggers
- `/pantry` (explicit skill invocation)
- `/pantry add`
- `/pantry list`
- `/pantry import`
- `/pantry search`
- `/pantry plan`
- `/pantry recipes`
- `/pantry favorite`

## Trigger Phrases Examples
- "Add ingredients to my pantry"
- "Show my pantry"
- "Import a recipe from..."
- "Plan meals for this week"
- "Generate a meal plan"
- "What's in my pantry?"
- "Search for recipes"
- "Find recipes using..."

## Context Clues
The skill should also activate when:
- User mentions organizing groceries or ingredients
- User asks about meal planning with ingredient efficiency
- User wants to import recipes from URLs
- User discusses reducing food waste

## Exclusions
Do NOT trigger this skill for:
- General cooking questions unrelated to planning
- Recipe discussions without pantry/planning context
- Food ordering or restaurant recommendations
